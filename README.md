SIMADUN (Sistem Informasi Manajemen Aduan & Tugas Akhir)
SIMADUN adalah platform berbasis web yang dirancang untuk mengelola proses administrasi akademik di lingkungan Departemen Teknik Elektro dan Informatika. Sistem ini bertujuan untuk mendigitalisasi alur pengajuan, pengujian, dan manajemen data mahasiswa agar lebih terintegrasi dan transparan.

🚀 Fitur Utama
Manajemen Penguji (Update): Implementasi fitur Penguji 3 untuk mengakomodasi kebutuhan penguji tambahan dalam sidang atau evaluasi mahasiswa.

Integrasi Basis Data: Sinkronisasi database lokal dengan database utama Departemen Teknik Elektro dan Informatika untuk memastikan data mahasiswa valid dan real-time.

Manajemen Tugas Akhir: Pelacakan status pengajuan tugas akhir dari awal hingga selesai.

Sistem Akuntabilitas: Mempermudah pelaporan dan dokumentasi kegiatan organisasi/akademik di lingkup Fakultas Teknik.

🛠️ Teknologi yang Digunakan
Framework: Laravel (PHP)

Database: MySQL

Frontend: Blade Template Engine, Bootstrap/Tailwind CSS

Tools: Git, Composer, XAMPP/Laragon

📂 Struktur Proyek & Perubahan Terbaru
Proyek ini telah melalui tahap optimasi dan penggabungan kelayakan sistem, termasuk:

Modul Penilaian: Penambahan logika baru untuk keterlibatan tiga penguji dalam satu sesi.

Skema Database: Pembaruan tabel relasi untuk mendukung integrasi lintas departemen.

UI/UX: Penyesuaian antarmuka untuk fitur input nilai penguji tambahan.
